# 원두 자동 동기화 시스템 구현 체크리스트

이 문서는 커피 원두 자동 동기화 시스템 구현을 위한 단계별 체크리스트와 GitHub 커밋 계획을 제공합니다.

## 🎯 프로젝트 개요

이 프로젝트는 다양한 커피 브랜드(카페) 웹사이트를 크롤링하여 원두 정보를 자동으로 수집하고, Firebase Firestore에 저장하여 웹 애플리케이션에 반영하는 시스템을 구축합니다.
